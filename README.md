BrajBhoomi Tours & Travels — Modern Website (HTML + Tailwind CSS)

This project is a fully responsive, modern, premium-looking website created for BrajBhoomi Tours & Travels using only HTML and Tailwind CSS.
The goal of the project is to redesign the original website into a cleaner, more professional, and more elegant Black-and-Gold themed interface while maintaining fast performance and a seamless user experience.

✨ Features
✔ Modern Premium UI

Black + Gold luxury theme

Glassmorphism effects

Soft shadows and rounded gradients

Clean typography and spacing

✔ Fully Responsive

Mobile-first design

Smooth layout for all devices (phones, tablets, desktops)

✔ Multi-Page Website

The project includes:

Home (index.html)

About Us (about.html)

Packages (packages.html)

Contact / Enquiry (contact.html)

All pages built using only HTML + Tailwind CSS CDN (no frameworks, no JS build tools).

✔ Enquiry Form (FormSubmit Integration)

The contact form is integrated with FormSubmit, allowing all form details to be sent directly to Gmail without backend code.

Sends Name, Phone, Email, Package selected, and Message

Uses _subject, _template, and _next FormSubmit options

Fully functional with redirect support

✔ WhatsApp Contact Buttons

Floating WhatsApp button on all pages

WhatsApp “Contact Now” button inside Enquiry page

Uses WhatsApp Click-to-Chat API (https://wa.me/)
